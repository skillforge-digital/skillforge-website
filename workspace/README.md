# SkillForge Workspace

This folder contains a printable HTML workspace prepared from the public SkillForge website and portal repositories.

## Files

- `index.html` - master workspace home
- `organization-structure.html` - board-level org design
- `brand-identity.html` - positioning, architecture, authority model
- `audience-segment-codes.html` - audience coding system for messaging
- `social-responsibility-matrix.html` - platform/account ownership and responsibility codes
- `content-operations.html` - daily posting and content operating system
- `organizational-development.html` - staffing and capability development roadmap
